Estructura de MyBase:



La aplicación MyBase cuenta con tres partes principales, las cuales son:

  1. La barra de menús.
  2. El panel Explorer.
  3. El panel Editor.

Dentro de la barra de menús se pueden encontrar opciones como crear nuevo proyecto, abrir un nuevo proyecto, etc. El panel Explorer es el encargado de visualizar la estructura de la base de datos. En él se muestra la base de datos y su contenido (tablas y campos), además de un menú contextual que permite realizar una serie de operaciones, las se ven más abajo. Por último tenemos el panel Editor, en el cual se realizan todas las operaciones sobre la base de datos, por ejemplo la creación de nuevas tablas, nuevos campos y la edición de los mismos, etc. Cada una de las operaciones y funcionalidades que brindan estas partes se verán en las páginas siguientes cuando se vaya describiendo el sistema.

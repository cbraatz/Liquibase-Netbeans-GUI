Abrir un proyecto existente:

Así cómo es posible crear un proyecto nuevo también es posible abrir un proyecto ya existente. A continuación se explica cómo realizar esta operación:



Lo primero que se debe realizar es acceder al menú Archivo de la barra de menús y seleccionar la opción Abrir Proyecto, los cual hará que se visualice la siguiente ventana.



Una vez que aparezca esta ventana es necesario especificar la dirección donde se encuentre el proyecto, para ello es requiere presionar el botón que se encuentra resaltado en la imagen. Esta acción hará que aparezca un “FileChooser” o selector de archivos, a través del cual es posible especificar la dirección del proyecto.

Cuando se haya encontrado y seleccionado el proyecto, el botón Finalizar de la ventana anterior se activará. Presionando este botón el proyecto se creará y se finalizará la operación.
